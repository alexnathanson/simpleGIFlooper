# Simple GIF Looper

## Setup
* sudo apt-get install libdbus-1-dev
* pip install omxplayer
* pip install omxplayer-wrapper




### not necessary...
* sudo apt-get install imagemagick

* sudo apt-get install ffmpeg


### Miscelanous commands
Play a video
* omxplayer FILE_NAME
  
Loop a single video
* omxplayer --loop FILE_NAME
